GitHub is a highly used software that is typically used for version control. It is helpful when more than just one person is working on a project.
Github document the changes and reflect them in an organized manner to avoid any chaos between any of the files uploaded.

Development of the GitHub.com platform began on October 19, 2007. The site was launched in April 2008 by Tom Preston-Werner, Chris Wanstrath, P. J. Hyett and Scott Chacon after it had been made available for a few months prior as a beta release.
